Projekt: Einfacher Taschenrechner
1. Stunde:

1.1. Grundlegende Struktur und Eingabe:

Erstellen Sie eine neue Java-Klasse namens Taschenrechner.
Implementieren Sie eine Methode main, in der Sie den Benutzer nach zwei Zahlen und einer Operation (z.B. +, -, *, /) fragen.
Verwenden Sie Scanner, um die Eingaben des Benutzers zu lesen.
1.2. Operationen:

Implementieren Sie separate Methoden für jede der vier Grundrechenarten: addieren, subtrahieren, multiplizieren und dividieren.
Jede Methode sollte zwei double-Parameter akzeptieren und das Ergebnis zurückgeben.
1.3. Bedingte Anweisungen:

Verwenden Sie if-else-Anweisungen, um zu bestimmen, welche Methode basierend auf der vom Benutzer eingegebenen Operation aufgerufen werden soll.
1.4. Fehlerbehandlung:

Fügen Sie eine Überprüfung hinzu, um sicherzustellen, dass der Benutzer nicht durch Null teilt.
Geben Sie eine Fehlermeldung aus, wenn der Benutzer eine ungültige Operation eingibt.
2. Stunde:

2.1. Erweiterte Funktionen:

Fügen Sie eine Methode hinzu, die den Durchschnitt eines Arrays von Zahlen berechnet.
Fragen Sie den Benutzer, wie viele Zahlen er eingeben möchte, und lesen Sie dann diese Zahlen in ein Array ein.
2.2. Schleifen:

Verwenden Sie eine for-Schleife, um durch das Array zu iterieren und den Durchschnitt zu berechnen.
Implementieren Sie eine weitere Methode, die das Array mit einer foreach-Schleife ausgibt.
2.3. Testing:

Erstellen Sie eine separate Klasse für JUnit-Tests.
Schreiben Sie Tests für jede Ihrer Methoden, insbesondere für Randfälle (z.B. Division durch Null).
2.4. Bonus:

Erweitern Sie Ihren Taschenrechner, um weitere Funktionen wie Quadratwurzel, Potenzieren oder Modulo zu unterstützen.
Fügen Sie eine Schleife hinzu, die den Benutzer fragt, ob er eine weitere Berechnung durchführen möchte, und führen Sie das Programm so lange aus, bis der Benutzer "nein" eingibt.
Viel Spaß beim Programmieren! Wenn Sie Fragen haben oder weitere Anleitungen benötigen, lassen Sie es mich wissen.
